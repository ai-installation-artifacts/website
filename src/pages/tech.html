TECH

The installation runs on a locally hosted infrastructure. All data processing happens in real time. A local AI model generates personalized documents based on minimal user input. These documents are printed immediately, shredded optionally, and never stored beyond the session.

The interaction flow is designed for immediacy and irreversibility. Inputs are processed, responses are physicalized. There is no back button and no retrieval.

The system uses an iPad interface for data entry, networked with a local server running a custom backend. Documents are generated using a LaTeX-based pipeline, compiled on-site, and printed through standard office hardware.

Everything appears functional, but beneath the surface, complex processes unfold. Just like in real systems: the interface is simple, the implications are not.
